该项目仅供学习使用，不提供任何安装包，开源协议 GPLv2

## 运行

- 请使用 fvm 运行该项目,版本限制 3.6.0 到 3.6.2，3.7.0 以上模拟器模糊效果会造成闪退
- 由于 ffmpeg-kit 停止维护了，这里使用了两个其他修复源，请根据运行环境修改 pubspec.yaml
